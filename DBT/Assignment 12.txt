Assignment – 12
Using the operators IN, ANY, and ALL.

1) Write a query that selects all customers whose ratings are equal to or greater than
ANY of Serres’.
Ans: SELECT * FROM CUSTOMERS 
WHERE  Rating >= (SELECT Snum FROM SALESPEOPLE
                 WHERE  Sname = 'Serres');

2) Write a query using ANY or ALL that will find all salespeople who have no
customers located in their city.
Ans: SELECT * FROM SALESPEOPLE 
WHERE City != ALL (SELECT City FROM CUSTOMERS);

3) Write a query that selects all orders for amounts greater than any for the
customers in London.
Ans: SELECT * FROM ORDERS 
WHERE Amt > ANY (SELECT Amt FROM ORDERS
                 WHERE Cnum IN (SELECT Cnum FROM CUSTOMERS
                               WHERE City = 'London'));

4) Write the above query using MIN or MAX.
Ans: SELECT * FROM ORDERS
WHERE Amt > ANY (SELECT MAX(Amt) FROM ORDERS
                 WHERE Cnum IN (SELECT Cnum FROM CUSTOMERS
                               WHERE City = 'London'));

SELECT * FROM ORDERS
WHERE Amt > ANY (SELECT MIN(Amt) FROM ORDERS
                 WHERE Cnum IN (SELECT Cnum FROM CUSTOMERS
                               WHERE City = 'London'));
