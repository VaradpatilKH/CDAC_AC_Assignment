# OOP using C++ – Assignment 2

This repository contains solutions for **Assignment 2** of **Object Oriented Programming using C++**.

## Files

* `Q1_Box.cpp` – Box class with constructor and volume calculation
* `Q2_SwapMethods.cpp` – Swap by value, address, and reference
* `Q3_PointerReference.cpp` – Pointer and reference demonstration
* `Q4_NewArray.cpp` – Dynamic array using `new` and `delete[]`
* `Q4_MallocArray.cpp` – Dynamic array using `malloc` and `free`
* `Q5_StudentConstructors.cpp` – Student class with constructors
* `Q6_EmployeeInitializer.cpp` – Employee class using initializer list
* `Theory_Answers.md` – Answers to theory questions
